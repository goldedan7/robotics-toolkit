# robotics-toolkit

A personal collection of robotics fundamentals implemented from scratch in Python.
Built as part of my journey from Mechanical Engineering into Physical AI & Robotics.

## Status
🚧 Active development — Phase 1 (Foundations)

## Roadmap

### Module 1: 2D Manipulator Kinematics (current)
- [ ] Forward kinematics for 2-link planar arm
- [ ] Visualization with matplotlib
- [ ] Jacobian computation
- [ ] Analytical inverse kinematics
- [ ] Unit tests

### Future modules
- 3D kinematics (DH parameters)
- Rigid body dynamics
- PID and trajectory control
- Simple simulation environments

## Tech Stack
- Python 3.11+
- NumPy, matplotlib
- pytest (testing)

## About
Mechanical Engineering graduate (University of Bristol, 2025) currently completing
military service in South Korea. This repo tracks my technical re-skilling toward
robotics and manipulation research, with the goal of pursuing an MSc in Robotics.
